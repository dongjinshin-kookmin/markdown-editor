# 마크다운 에디터

브라우저에서 바로 쓰는 **단독실행형(single-file) 마크다운 에디터**입니다.
외부 라이브러리·CDN·네트워크 의존성이 전혀 없어 `index.html` 하나만 있으면
오프라인에서도 더블클릭으로 동작합니다.

**라이브 데모:** https://dongjinshin-kookmin.github.io/markdown-editor/

## 기능

- **분할 라이브 프리뷰** — 에디터만 / 분할 / 프리뷰만 3모드, 드래그 리사이저, 스크롤 동기화
- **자체 마크다운 파서** — 제목, 강조/취소선, 코드 블록(신택스 하이라이팅: JS·Python·HTML·CSS·SQL), 중첩 인용·목록, 체크리스트, GFM 테이블(정렬), 링크/이미지, 수평선
- **XSS 방지** — raw HTML 전면 이스케이프, 위험 URL 스킴 차단
- **툴바 15종 + 단축키** — Cmd/Ctrl+B(굵게)·I(기울임)·K(링크)·E(코드)·S(저장) 등
- **파일 입출력** — .md 열기(선택·드래그앤드롭), .md 저장, 스타일 포함 HTML 내보내기, 프리뷰 인쇄
- **다중 문서 자동저장** — localStorage 기반 문서 목록(새로/이름변경/삭제/전환), 커서·테마·뷰 상태 복원
- **다크/라이트 테마** — 시스템 설정 기본값 추종
- **에디터 UX** — Tab 들여쓰기(다중행·Shift 내어쓰기), 목록에서 Enter 시 마커 자동 이어쓰기

## 사용

로컬: `index.html`을 브라우저로 열기 — 끝.

## 제한사항

- 문서는 브라우저 localStorage에 저장됩니다. 브라우저 데이터를 삭제하면 사라지므로 중요한 문서는 `.md`로 내려받아 보관하세요.
- 툴바/단축키로 넣은 편집은 브라우저에 따라 Cmd/Ctrl+Z 실행취소가 안 될 수 있습니다(일반 타이핑은 정상).
- CommonMark 전수 호환은 아닙니다 — setext 제목, 참조형 링크, 각주는 미지원.
